# text-to-speech-app
Type something and speech out. <br>
Simple text to voice app made with JS. Heard a voice talking everything you're written before.

Technologies used:

<li>HTML</li>
<li>CSS</li>
<li>Javascript</li>
<li>Google Fonts</li>
